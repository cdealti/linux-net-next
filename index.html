Unofficial patch set(s) for the Linux Wireless Sensor Network project

What is this project?

This project aims to provide the latest bugfixes on both the IEEE 802.15.4 and the 6LoWPAN stacks within the Linux Kernel to a wider audience.

On the long term, these patches should be incorporated to the net-next branch of the Linux kernel. This page aims to regroup all of them, and provide them a bigger exposure, for more extensive reviews.

Why does this project exists?

While an fair number of very motivated persons work on the Linux IEEE 802.15.4 and 6LoWPAN stacks, most of them have very little time to actually push their work to the net-next branch of the Linux kernel for inclusion, or seek for more review before they do so. Consequently, a newcomer to the project will spend a lot of time figuring out where the most recent work is located, because it is scattered in so many places.

This project aims to tackle this particular issue, and to make it easier to newcomers to have a recent and functional version of the kernel, benefiting from the latest bugfixes.

How to use

In order to retrieve a recent version of the net-next branch of the Linux kernel and apply the patches on top of it, you need to perform the two following step:

  • clone this repository
  • checkout the branch your interested in (e.g. git checkout 6lowpan)

How to contribute

The proper way to get your patches reviewed is to send them to the Linux Zigbee mailing list (linux-zigbee-devel@lists.sourceforge.net). You can find more on the list (including the archives) here.

If I see your patch(es) on the lists and find it sensible (yes, that's highly suggestive), I will include it in the sets here. However if I fail to notice your patch, you can either send me a pull request, or your patches directly.
